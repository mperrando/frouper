# Frouper

Aggregates a serie of timed values slicing them in time slots
and returns statisctical values for each slot.

## Installation

Add this line to your application's Gemfile:

    gem 'frouper'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install frouper
